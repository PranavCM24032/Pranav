HTML Tutorial

HTML (Hypertext Markup Language) is the standard language used to create and design web pages. HTML describes the structure of web pages using elements or "tags". It allows us to structure content like headings, paragraphs, links, images, tables, forms, and more.

Why Use HTML?

  • HTML is the backbone of any website. It provides the basic structure for web pages.
  • Without HTML, web browsers wouldn't know how to display content like text, images, or links.
  • HTML enables us to create interactive and multimedia-rich websites when combined with CSS and JavaScript.

Basic HTML Example

Here is a very simple HTML page structure:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Simple HTML Page</title>
  </head>
  <body>
    <h1>Welcome to HTML Tutorial</h1>
    <p>This is a basic example of an HTML page.</p>
    <a href="https://www.example.com">Click here to visit Example</a>
  </body>
</html>

Explanation of the HTML Code

  • <!DOCTYPE html>: This declaration defines the document as HTML5. It tells the browser which version of HTML to use.
  • <html>: This is the root element of the HTML page. It wraps all the content of the page.
  • <head>: The <head> element contains metadata about the document, like the title, character encoding, and link to stylesheets.
  • <meta charset="UTF-8">: This tag defines the character encoding for the page, ensuring that text displays correctly in all languages.
  • <body>: The <body> contains all the visible content of the page, like text, images, links, etc.
  • <h1>: The <h1> tag defines the main heading of the page. It is typically the most important heading.
  • <p>: The <p> tag is used for paragraphs of text.
  • <a href="URL">: The <a> tag defines a hyperlink. The <href> attribute specifies the link destination.

Key HTML Elements

  • <h1> to <h6>: These tags define headings. <h1> is the largest, and <h6> is the smallest.
  • <p>: Used to define a paragraph of text.
  • <a>: Defines a hyperlink to another page or a resource.
  • <img>: Embeds an image in the webpage. Example: <img src="image.jpg" alt="Image Description">
  • <ul> and <ol>: Used for unordered (bulleted) and ordered (numbered) lists, respectively.
  • <div>: Defines a division or section of a page. It's used to group content for styling or scripting.
