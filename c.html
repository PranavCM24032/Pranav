C Language Tutorial

C is a general-purpose programming language that is widely used for system programming, embedded systems, and software development. It was developed by Dennis Ritchie at AT&T Bell Labs in 1972. C is known for its efficiency, speed, and close-to-hardware programming capabilities.

Basic Example - "Hello, World!"

This is a simple "Hello, World!" program in C. It's the most basic program you'll write as a beginner.

#include

int main() {
printf("Hello, World!\\n");
return 0;
}

Explanation:

  • #include - Includes the standard input/output header file for using functions like printf().
  • int main() - Defines the main function, the entry point of the program.
  • printf("Hello, World!"); - Outputs the message "Hello, World!" to the console.
  • return 0; - Indicates successful completion of the program.
