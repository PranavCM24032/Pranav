CSS Tutorial

CSS (Cascading Style Sheets) is used to style HTML pages â€” colors, fonts, spacing, and layout.

Why Use CSS?

  • Separates content from design
  • Makes websites responsive
  • Easy to maintain and scale

Types of CSS

  • Inline CSS: Added directly to HTML tags
  • Internal CSS: Written in a <style> tag inside <head>
  • External CSS: Linked through a .css file

Example Code

/* styles.css */
body {
  background-color: #f0f0f0;
}

h1 {
  color: #333;
}

p {
  color: #555;
  font-size: 16px;
}
